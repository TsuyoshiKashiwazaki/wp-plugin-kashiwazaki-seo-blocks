# Kashiwazaki SEO Blocks

WordPressのブロックエディタ（Gutenberg）用の高機能カスタムブロックコレクションプラグインです。

## 概要

Kashiwazaki SEO Blocksは、SEOに配慮したコンテンツ作成を支援するWordPressプラグインです。美しいデザインと高いカスタマイズ性を持つ複数のブロックを提供し、プロフェッショナルなウェブサイト制作を強力にサポートします。

## 主な特徴

- 🎨 **豊富なカスタマイズオプション** - 色、サイズ、スタイルを自由に調整
- 📱 **レスポンシブ対応** - モバイルからデスクトップまで美しく表示
- 🚀 **軽量で高速** - パフォーマンスを重視した設計
- 🔧 **簡単操作** - 直感的なインターフェースで誰でも使いやすい
- 🎯 **SEO最適化** - 構造化された見出しとコンテンツ配置
- 🇯🇵 **完全日本語対応** - メニューから設定まですべて日本語

## 利用可能なブロック

### 1. 見出しバー（Heading Bar）
SEOに重要な見出しタグを美しく装飾
- **H1〜H6**の全見出しレベルに対応
- 背景色、文字色、枠線のカスタマイズ
- 影効果と角丸の設定
- パディング・マージンの細かい調整

### 2. ボタン（Button）
クリック率を高める魅力的なボタン
- リンク付きの装飾ボタン
- 新規タブで開く設定
- 全幅表示オプション
- ホバーエフェクト対応
- 豊富なカラーバリエーション

### 3. 汎用ボックス（Universal Box）
情報を効果的に伝えるコンテンツボックス
- 5種類のアイコンタイプ（情報、警告、成功、注意、ヒント）
- グラデーション背景対応
- 枠線スタイル（なし、実線、点線、破線、二重線）
- ネスト可能なInnerBlocks対応
- 影効果のカスタマイズ

### 4. ボーダー（Border）
コンテンツを区切る装飾的な境界線
- 線の太さ、色、スタイルをカスタマイズ
- 4種類の線スタイル（実線、点線、破線、二重線）
- マージン調整可能
- グラデーション対応

### 5. テーブル（Table）
データを見やすく表示するレスポンシブテーブル
- 動的な行・列の追加削除
- ヘッダー行のオン/オフ
- ストライプ模様とホバー効果
- 横スクロール機能（レスポンシブ対応）
- セル内のRichText編集
- 配置（左揃え、中央揃え、右揃え）

### 6. 吹き出し（Speech Bubble）
会話形式でコンテンツを表現
- アバター画像の設定（円形・四角形）
- 左右の配置選択
- 通常の吹き出しと考え事スタイル
- 枠線の太さに応じた尻尾の自動調整
- 吹き出し幅のカスタマイズ（0-100%）
- 名前表示オプション

## インストール方法

1. プラグインフォルダを`/wp-content/plugins/`にアップロード
2. WordPress管理画面の「プラグイン」メニューから有効化
3. すぐに利用開始できます

## 使い方

### 基本的な使用方法
1. 投稿または固定ページの編集画面を開く
2. ブロック追加ボタン（＋）をクリック
3. 「Kashiwazaki SEO Blocks」カテゴリーを選択
4. 使用したいブロックを選択
5. コンテンツを入力
6. 右側のインスペクターパネルで詳細設定をカスタマイズ

### 各ブロックの活用例

#### 見出しバーの活用
- 記事の大見出しとして使用
- セクションの区切りに使用
- 重要な情報を強調

#### ボタンの活用
- CTAボタンとして使用
- ダウンロードリンク
- 外部サイトへの誘導

#### 汎用ボックスの活用
- 注意事項の表示（警告アイコン）
- ヒントやコツの紹介（ヒントアイコン）
- 成功メッセージの表示（成功アイコン）

#### テーブルの活用
- 料金表の作成
- 仕様一覧の表示
- 比較表の作成

#### 吹き出しの活用
- FAQ形式のコンテンツ
- インタビュー記事
- キャラクターによる解説

## システム要件

- **WordPress**: 6.0以上
- **PHP**: 7.4以上
- **ブラウザ**: Chrome、Firefox、Safari、Edge（最新版）

## トラブルシューティング

### ブロックが表示されない場合
1. プラグインが有効化されているか確認
2. WordPress、PHPのバージョンを確認
3. ブラウザのキャッシュをクリア

### スタイルが適用されない場合
1. テーマとの競合を確認
2. 他のプラグインとの競合を確認
3. ブラウザの開発者ツールでエラーを確認

## よくある質問

**Q: 既存のコンテンツに影響はありますか？**
A: いいえ、既存のコンテンツには影響しません。

**Q: クラシックエディタでも使えますか？**
A: いいえ、ブロックエディタ（Gutenberg）専用です。

**Q: カスタムCSSは追加できますか？**
A: はい、各ブロックにカスタムクラスを追加できます。
