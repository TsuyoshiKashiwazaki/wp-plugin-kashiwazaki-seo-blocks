SEO最適化された高機能カスタムブロック集。直感的な操作で色・サイズ・スタイルを細かくカスタマイズ可能。

== Description ==

Kashiwazaki SEO Blocksは、WordPressのブロックエディタ（Gutenberg）用のカスタムブロックコレクションです。SEOに配慮したコンテンツ作成を支援し、美しいデザインと高いカスタマイズ性を提供します。

= 利用可能なブロック =

* **見出しバー** - H1〜H6まで選択可能な装飾付き見出し
* **ボタン** - リンク付きのカスタマイズ可能なボタン
* **汎用ボックス** - アイコン付きボックス（情報、警告、成功、ヒント等）
* **ボーダー** - ページ区切り用の区切り線
* **テーブル** - レスポンシブ対応のテーブル
* **吹き出し** - アバター画像付きの会話形式コンテンツ

= 主な機能 =

* 全ブロックで背景色、文字色、枠線をカスタマイズ可能
* 余白（パディング・マージン）の細かい調整
* 影の効果や角丸の設定
* レスポンシブデザイン対応
* 汎用ボックスの最大幅を%またはpxで選択可能
* 汎用ボックスの配置（左寄せ/右寄せ/センタリング）
* グラデーション背景のサポート

== Installation ==

1. プラグインをダウンロード
2. WordPressの管理画面から「プラグイン」→「新規追加」→「プラグインのアップロード」
3. ダウンロードしたZIPファイルを選択してインストール
4. プラグインを有効化
