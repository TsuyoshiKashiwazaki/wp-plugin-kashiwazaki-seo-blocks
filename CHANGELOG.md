# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2025-12-04

### Added
- 管理画面に「ブロック使用状況」タブを新設
- 吹き出しプリセット管理機能（登録・編集・削除・適用）
- ボタンプリセット管理機能（登録・編集・削除・適用）
- 汎用ボックスプリセット管理機能（登録・編集・削除・適用）
- 吹き出しの一括変換機能（プリセット間での変換）
- ボタンの一括変換機能
- 汎用ボックスの一括変換機能
- ブロックエディター内でのプリセット保存・読み込み機能

### Improved
- 管理画面「このプラグインについて」タブの内容を充実

## [1.0.1] - 2025-11-20

### Added
- 汎用ボックスに最大幅設定機能を追加（px/%選択可能）
- 汎用ボックスに配置機能を追加（左寄せ/右寄せ/センタリング）

### Improved
- 汎用ボックスの幅をmax-widthで実装し、レスポンシブ対応を強化
- 単位切り替え時に適切な値範囲へ自動調整

## [1.0.0] - 2025-09-11

### Added
- Initial release of Kashiwazaki SEO Blocks plugin
- Heading Bar block with H1-H6 support and customizable styles
- Button block with link support and full-width option
- Generic Box block with icon support (info, warning, success, alert, tip)
- Border block for page sections
- Table block with responsive design and various styling options
- Speech Bubble block with avatar support
- Custom block category "Kashiwazaki SEO Blocks"
- Comprehensive customization options for all blocks (colors, borders, padding, margins, shadows)
- Gradient background support for boxes
- Responsive design support
- Admin settings page with plugin information

### Features
- WordPress 6.0+ compatibility
- PHP 7.4+ compatibility
- SEO-optimized block structure
- Intuitive user interface
- Professional styling options
